NAME

ipsec _plutoload - internal script to start pluto  

DESCRIPTION

_plutoload is called by _plutorun to actually start the pluto executable.  

SEE ALSO

ipsec(8), ipsec_setup(8), ipsec__realsetup(8), ipsec__plutorun(8)
